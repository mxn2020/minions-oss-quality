# Minions Oss-quality

Linting rules, test coverage targets, usability checklists, and release gates

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-oss-quality/sdk`) — Core library for Node.js
- **Python SDK** (`minions-oss-quality`) — Python bindings
- **CLI Tool** (`@minions-oss-quality/cli`) — Command-line interface
- **Web App** — Interactive playground at [oss-quality.minions.wtf](https://oss-quality.minions.wtf)
